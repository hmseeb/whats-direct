## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/hmseeb/whats-direct.git
    ```

2. Navigate to the project directory:
    ```bash
    cd whats-direct
    ```

3. Install the dependencies:
    ```bash
    flutter pub get
    ```

4. Run the app:
    ```bash
    flutter run
    ```
